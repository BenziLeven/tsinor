# tsinor
Store Class Assignment

## Installation guide

### Clone the repo
```
git clone https://github.com/BenziLeven/tsinor.git
```
### install dependencies
```
yarn install
```
or
```
npm i
```
### Create data folder (required)
```
cd tsinor
mkdir data
```

### run the tests

```
yarn test
```
or
```
npm run test
```

### run the examlpe script

```
yarn start
```
or
```
npm run start
```
